1) How to create GitHub repository

Answer-)
1.Go to github.com and log in.

2.Click the “New” or “+ → New Repository” button.

3.Give your repo a name.

4.Choose Public or Private.

5.Check the option “Add a README file”.

6.Click Create Repository.



2) How to run js program using node js ?

Answer-)

1. Create a dev container.
2. install node js & js script 
3. select version 18 bookworm.
4. click ok.
5. create a folder inside dev container .
6. create a file inside folder with .js extension.
7. Run the file using command node filepath.


3) How to commit and push code?

Answer-)

1. Go to source control.
2. select add changes(+).
3. give commit message .
4. click on commit .
5. Then click on sync chnages.
6. Push the code.
